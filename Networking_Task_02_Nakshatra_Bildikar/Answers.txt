# Networking Task 02 

Part A: Network Devices Research
1. Router
Ans : A router is a networking device that connects different networks together
Purpose : 
main purpose is to direct data packets between networks and provide internet access to multiple devices 
Working : 
router receives data from a device, checks the destination IP address, and forwards the data to the correct network 
Usage : 
Wifi and routers at homes and offices etc

2. Switch
Ans : A device that connects multiple devices within the same local area network (LAN)
Purpose : 
allows electronic devices to communicate efficiently 
Working : 
A switch gets the MAC address of connected devices and sends data only to the listed device instead of broadcasting it to everyone
Usage : 
School computer Labs 

3. Hub 
Ans : Basic networking device used to connected multiple devices in a network
Purpose : 
allows devices to share network communication 
Working : 
Hub receives data from a device and sends the same data to every connected device , 
regardless of the targeted or expected receiver 
Usage : 
Testing environments 

4. Access Point (AP)
Ans : A device that provides wireless connectivity to devices in a network 
Purpose : 
connecting wireless devices to a wired device using wifi 
Working : 
AP receives the data fromm wired network and transmits it as wireless signals , also receives signals from the devices and forwards then to the network 
Usage : 
Airports , malls 

5. Firewall 
Ans : Security software which monitors and controls the traffic 
Purpose : 
protecting networks and devices from malware , unauthorized access and cyberattacks 
Working : 
analyzes incoming and outgoing data packets and allows or rejects them acc to security rules 
Usage :
Banking systems 

6. Modern 
Ans : Networking device that connects a user's network to an Internet Service Provider (ISP)
Purpose : 
establishes internet connectivity by converting signals between the ISP and user devices
Working : 
modem receives signals from the ISP, converts them into digital data for devices, and converts outgoing digital data into signals that can travel through communication lines
Usage : 
Broadband connections

Q2. Public and private ip add 
Ans : Public ip addresses - 1.1.1.1 , 8.8.8.8 
      Private ip addresses - 192.168.1.10 , 10.0.0.5 , 172.16.5.20 , 192.168.100.1 

We distinguish the above ip add based on their numbers ,  for private ip's theres a specified range of numbers like 10.0.0.0–10.255.255.255, 172.16.0.0–172.31.255.255, or 192.168.0.0–192.168.255.255. Any IP address outside these ranges is generally considered a Public IP address.
