Programming Task 04 - Functions & File Handling

Objective : 
This project demonstrates the use of functions, modular programming, file handling, and basic data management using Python.

Files Included

calculator_functions.py : 
A menu-driven calculator using functions for addition, subtraction, multiplication, and division.

student_manager.py : 
Accepts student details and displays formatted output using functions.

marks_analyzer.py : 
Calculates total marks, percentage, and grade based on student marks.

file_handling.py : 
Stores student data into a file and reads it back.

password_vault.py :
A simple password storage system that saves website credentials into a file.

Concepts Used :
Functions
Loops (for, while)
Conditional statements
File handling (read/write/append)
Basic data storage
Modular programming

Learning Outcome :
This task helped in understanding how to structure programs using functions, handle file operations, and simulate real-world systems like student records and password storage.
