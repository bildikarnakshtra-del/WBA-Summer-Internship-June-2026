Programming Task 03 - Loops, Patterns & Automation

Objective:

This project demonstrates the use of loops, conditional statements, and basic automation using Python.
Files Included

1. multiplication_table.py
Generates multiplication table for a given number up to 10.

2. number_analysis.py
Calculates:
Sum of numbers from 1 to N
Count of even numbers
Count of odd numbers

3. patterns.py
Prints different star and number patterns using loops.

4. password_attempt.py
Simulates a login system with 3 attempts and account lock feature.

5. username_generator.py
Generates multiple username suggestions based on user input.

Concepts Used :
for loops
while loops
nested loops
conditional statements
string manipulation
basic automation logic
Learning Outcome

This task helped in understanding iteration, logic building, and basic problem-solving used in real-world programming and cybersecurity applications
