Part B : Purpose of all commands 
pwd : Displays the current working directory path
ls : Lists files and folders in the current directory
ls -la : Lists all files and folders, including hidden files, with detailed information
cd : Changes the current directory
clear : Clears the terminal screen
history : Shows previously executed commands
whoami : Displays the current logged-in username
hostname : Displays the system's host/computer name

Part D : What do all the used commands do 
touch : creates a nwe empty file , if file already exists then updates the timestamp 
cp : copies a file from one loc to another 
mv : moves a file to different loc , can also be used to rename files 
rm : removes files from the system 

EXAMPLES USED 
touch notes.txt : created an empty file with the name notes.txt 
touch commands.txt : created an empty file named commands.txt
touch report.txt : created an empty file named report.txt 
cp notes.txt Networking/ : copied notes.txt into Nteworking folder 
mv commands.txt Linux/ : moved commands.txt to linux folder

Part F: Linux Research Activity

1. What is Linux?
Linux is an open-source operating system based on Unix. It is widely used on computers, servers, cloud platforms, and cybersecurity systems because it is secure, stable, and flexible

2. Why is Linux important in Cyber Security?
Linux is important in cybersecurity because many servers and security tools run on Linux. It provides powerful command-line utilities, better system control, and strong security features that help security professionals perform monitoring, testing, and administration tasks

3. Difference between Linux and Windows
Linux:
- Open-source operating system
- Free to use
- Highly customizable
- Commonly used on servers and cybersecurity systems

Windows:
- Proprietary operating system developed by Microsoft
- Requires a license for many versions
- More commonly used on personal computers
- Easier for beginners

4. What is a Linux Distribution?
A Linux distribution is a version of Linux that includes the Linux kernel along with software packages, utilities, and a desktop environment. Examples include Ubuntu, Kali Linux, and Parrot OS

5. Why do Ethical Hackers prefer Linux-based Operating Systems?
Ethical hackers prefer Linux because it provides powerful command-line tools, strong networking capabilities, and extensive customization options. Many cybersecurity tools are designed for Linux, and distributions like Kali Linux come with security testing tools pre-installed
