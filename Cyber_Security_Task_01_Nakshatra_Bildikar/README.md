Part B – Password Security Review

1. What makes a strong password?
Ans. A strong password contains at least 12 characters and includes uppercase letters, lowercase letters, numbers, and special symbols.

2. Why should passwords be unique?
Ans. Unique passwords prevent attackers from accessing multiple accounts if one password is leaked.

3. What is MFA?
Ans. Multi-Factor Authentication is an additional security layer that requires another verification method besides a password.

4. Risks of password reuse?
Ans. If one account is hacked, attackers can use the same password on other accounts.

Password Policy (5 Best Practices)
Use at least 12 characters.
Include uppercase and lowercase letters.
Include numbers and special symbols.
Avoid personal information.
Enable MFA whenever possible.

Part D – Cyber Threat Research

1. Phishing
Definition
Phishing is a cyberattack where attackers impersonate trusted organizations to steal information.

Example
Fake bank email asking for login credentials.

Prevention
Verify sender
Don't click suspicious links
Enable MFA

2. Malware
Definition
Malware is malicious software designed to damage systems.

Example
Trojan Horse malware.

Prevention
Install antivirus
Download only from trusted sources
Update software

3. Ransomware
Definition
Ransomware encrypts files and demands payment.

Example
WannaCry attack.

Prevention
Regular backups
Updated OS
Avoid suspicious attachments

4. Social Engineering
Definition
Manipulating people into revealing confidential information.

Example
Fake tech-support calls.

Prevention
Verify identities
Never share passwords
Security awareness

5. Data Breach
Definition
Unauthorized access to sensitive information.

Example
Facebook data leak.

Prevention
Strong passwords
MFA
Regular monitoring

Part F – Security Reflection Report

During this security audit, I identified several cybersecurity risks in my digital habits. Although I regularly use the internet for studying, communication, and social media, I realized that some of my accounts did not have Multi-Factor Authentication (MFA) enabled. I also found that I occasionally reuse similar passwords across different platforms, which can increase the risk of unauthorized access if one account is compromised. Additionally, I do not always verify the authenticity of emails and messages before opening links, which could make me vulnerable to phishing attacks.

To improve my cybersecurity, I plan to implement several security measures immediately. First, I will enable MFA on all important accounts such as Google, LinkedIn, and other online services. Second, I will create strong and unique passwords for every account and consider using a password manager to store them securely. I will also keep my operating system, antivirus software, and applications updated to protect against newly discovered vulnerabilities. Furthermore, I will be more cautious when clicking links, downloading files, or sharing personal information online.

Cybersecurity is extremely important in today's digital world because a large amount of personal, academic, and financial information is stored online. Cyber attacks can lead to data theft, financial loss, identity theft, and privacy violations. As technology becomes a part of everyday life, individuals must develop safe online habits and remain aware of potential threats. Practicing good cybersecurity not only protects personal information but also contributes to a safer digital environment for everyone. By adopting secure practices, we can reduce risks and use technology more confidently and responsibly.
