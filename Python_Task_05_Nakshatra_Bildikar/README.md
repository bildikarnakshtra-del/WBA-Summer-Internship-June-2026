Programming Task 05

Objective
This task demonstrates the use of Python Lists for storing, processing, searching, and analyzing data.

Programs Included
1. Student Marks Analyzer

Stores marks of 5 students and displays:
Highest Marks
Lowest Marks
Average Marks

2. Employee Record Search
Searches for an employee name from a predefined list and displays whether the record exists.

3. Number Statistics Tool
Accepts 10 numbers and displays:
Largest Number
Smallest Number
Sum
Average
Count of Even Numbers
Count of Odd Numbers

4. Cyber Security Log Analyzer
Analyzes login attempts and displays:
Total Login Attempts
Successful Logins
Failed Logins

5. Blacklisted IP Checker
Checks whether a given IP address exists in a blacklist.

Technologies Used
Python 3
VS Code

Conclusion
This task helped in understanding lists, loops, searching, data processing, and basic cybersecurity-related data analysis.
